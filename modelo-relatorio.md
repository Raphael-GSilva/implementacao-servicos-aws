# Introdução

Este relatório apresenta uma proposta de implementação de serviços AWS para uma farmácia fictícia com o objetivo de modernizar sua infraestrutura tecnológica, reduzir custos operacionais e aumentar a disponibilidade dos sistemas.

A solução considera a utilização de serviços gerenciados da AWS para substituir parte da infraestrutura local, reduzindo custos com aquisição de equipamentos, manutenção e administração dos recursos computacionais.

---

# Descrição do Projeto

A implementação foi dividida em três etapas.

## Etapa 1 – AWS EC2

### Objetivo

Migrar os sistemas internos da empresa para instâncias virtuais.

### Caso de Uso

Hospedagem do sistema de gestão e controle de estoque.

**Benefícios**

- Eliminação de servidores físicos;
- Redução de custos com infraestrutura;
- Escalabilidade;
- Alta disponibilidade.

---

## Etapa 2 – AWS S3

### Objetivo

Centralizar o armazenamento de arquivos corporativos.

### Caso de Uso

Armazenamento de documentos fiscais, backups, imagens e relatórios.

**Benefícios**

- Alta durabilidade;
- Backup automatizado;
- Redução dos custos de armazenamento;
- Compartilhamento seguro de arquivos.

---

## Etapa 3 – AWS RDS

### Objetivo

Migrar o banco de dados para um serviço gerenciado.

### Caso de Uso

Armazenamento das informações de clientes, estoque, fornecedores e vendas.

**Benefícios**

- Backups automáticos;
- Alta disponibilidade;
- Escalabilidade;
- Redução da administração da infraestrutura do banco de dados.

---

# Benefícios Esperados

Com a implementação proposta espera-se:

- Redução dos custos operacionais;
- Modernização da infraestrutura;
- Maior disponibilidade dos sistemas;
- Segurança das informações;
- Escalabilidade;
- Automatização dos processos de backup.

---

# Conclusão

A adoção dos serviços AWS EC2, AWS S3 e AWS RDS proporciona uma infraestrutura moderna, segura e escalável.

A proposta reduz custos com infraestrutura física, simplifica a administração dos recursos computacionais e prepara a empresa para futuras expansões.
